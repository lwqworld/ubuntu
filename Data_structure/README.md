1、算法时间复杂度排序：

2、线性表
    (1)定义：由零个或多个数据元素组成的有限序列;
    (2)序列：即元素之间有先来后到的顺序;
    (3)数据类型：指一组性质相同的值的集合及定义在此集合上的一些操作的总称;
    (4)操作：
    InitLists(*L):初始化空表L
    ListEmpty(L):判断是否为空表
    ClearList(*L):清空
    GetElem(L,i,*e):将表L中第i个位置的元素值返回给e
    LocateElem(L,e):在表L中查找与e相等的元素，并返回表中序号
    ListInsert(*L,i,e):插入
    ListDelete(*L,i,*e):删除
    ListLength(L):返回表L的元素个数
